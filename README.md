# IntroIntoAI Projects

This workspace contains three AI-focused projects:

- `A-Apriori`: Apriori frequent itemset mining + association rules.
- `AStar`: A* pathfinding on a 10x10 Mindanao place grid.
- `EvolutionAlgo`: Genetic algorithm examples (Python maze solver + Unity project).

## Project Index

- `A-Apriori/README.md`
- `AStar/README.md`
- `EvolutionAlgo/README.md`

## Quick Start

Use your virtual environment, then run each project from its folder:

```powershell
# Apriori
cd A-Apriori
python Aprio.py

# A* pathfinding
cd ..\AStar
python Number2.py

# Evolutionary maze solver
cd ..\EvolutionAlgo
python maze.py
```
